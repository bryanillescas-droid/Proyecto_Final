Aplicación de Cotización de Seguros

Esta aplicación busca ayudar al usuario a automatizar el proceso de generar cotizaciones para la empresa TK-U.
La aplicación pedirá una lista de información que ayudará al sistema a determinar los valores correctos para la cotización.

Preguntas que la aplicación utilizará para la recolección de datos:

Ingrese su nombre, por favor.

¿Cuántos años tiene? Ingrese solamente números.

¿Está casado actualmente? (sí/no)

¿Qué edad tiene su esposo/a?

¿Tiene hijos o hijas? Indique la cantidad de hijos en números.

¿Tiene propiedades? Por favor, ingrese la cantidad en números.

Por favor, indique su salario mensual.

Dependiendo de la edad que tenga el usuario, se cobrará un porcentaje:

Entre 18 y 24 años, se cobra el 10%.

Entre 25 y 49 años, se cobra el 20%.

Si el usuario tiene 50 años o más, se cobra el 30%.

Si el usuario tiene cónyuge, este también genera un cobro dependiendo de la edad, aplicado sobre el precio base.
Si el usuario tiene hijos, se cobrará un 20% sobre el precio base por cada hijo.
Si el usuario tiene propiedades, se cobrará un 35% por cada propiedad.
Además, se cobrará el 5% del salario del usuario.

Esta aplicación continuará generando cotizaciones hasta que el usuario escriba la palabra "salir".
